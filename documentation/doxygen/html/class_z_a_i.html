Detailed Description

Defines a nucleus.

Define a nuclei as 3 integer Z,A,I:

  • its charge number : Z
  • its mass number : A
  • its Isomeric state : I (0 : fundamental, 1 : first isomeric state ,...)

The aim of this class is to discribe each ZAI.
